CSS: Cascading Style Sheets

CSS is the styling language used to control the appearance and layout of web pages. While HTML creates the structure, CSS improves how the page looks by controlling colours, fonts, spacing, positioning, animations, and responsiveness.

What Is CSS?

CSS stands for Cascading Style Sheets. The word cascading means that when more than one style rule applies to the same element, the browser decides which rule should be used based on importance, specificity, and order. CSS separates design from HTML structure, which makes websites easier to maintain and update.

CSS Selectors

Selectors are patterns used to target elements in the DOM so you can apply styles to them. They can match by element name, class, ID, attributes, position in the tree, state, and more.

Selector Example Example description
Type (element)pSelects all <p> elements.
Class.cardSelects elements with class="card".

Adjacent siblingh1 + pSelects the first <p> immediately after an <h1>.
General siblingh1 ~ pSelects all <p> siblings that follow an <h1>.
Attribute (exists)a[target]Selects links that have a target attribute.
Attribute (equals)input[type="email"]Inputs whose type is exactly email.
Pseudo-classa:hoverLinks when the mouse pointer hovers over them.
Pseudo-elementp::first-lineThe first formatted line of every paragraph.
Grouph1, h2, h3Selects all <h1>, <h2>, and <h3> elements.
Universal*Selects all elements (use sparingly).

Nth-childli:nth-child(2)The second list item within each parent list.
Nth-of-typep:nth-of-type(odd)Odd paragraphs within their parent.

CSS Functions

CSS functions are built-in helpers you can use inside property values for calculations, color creation/mixing, gradients, transforms, filters, responsive sizing, and more.

Function Description
attr(name)Uses an element attribute value (mainly in content and limited properties).
blur(radius)Filter that blurs an element by the given radius.
calc(expr)Performs math with units, e.g. width: calc(100% - 2rem).
clamp(min, pref, max)Clamps a value between a minimum and maximum, great for fluid sizing.
color-mix(in space, c1 p, c2)Mixes two colors in a color space, e.g. lab or srgb.
conic-gradient(...)Creates a gradient that sweeps around a center point.
contrast(amount)Filter that adjusts contrast of the element.
drop-shadow(...)Filter that applies a shadow following the element's alpha shape.
env(var)Reads environment variables, e.g. safe-area-inset-top on iOS.
hsl(h s% l% / a)Defines a color by hue, saturation, and lightness, with optional alpha.
invert(amount)Filter that inverts colors by the given amount.
linear-gradient(...)Creates a linear gradient image.
matrix(...)Low-level 2D transform matrix, rarely used directly.
max(a, b)Returns the larger of two or more values.
min(a, b)Returns the smaller of two or more values.
minmax(min, max)Grid track sizing function for grid-template-columns/rows.
radial-gradient(...)Creates a radial, circular, or elliptical gradient image.
rgb(r g b / a)Defines a color by red, green, and blue, with optional alpha.
rotate(angle)Transform that rotates an element.
saturate(amount)Filter that increases or decreases color saturation.
scale(x[, y])Transform that scales an element on x/y axes.
skew(ax[, ay])Transform that skews an element along axes.
translate(x[, y])Transform that moves an element along axes.
repeat(count, track)Grid repeat helper for track lists, e.g. repeat(3, 1fr).
url(path)References an external resource, such as images or fonts.
var(--name[, fallback])Reads a CSS custom property with optional fallback.
