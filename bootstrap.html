Bootstrap Framework

Bootstrap is a popular, open-source CSS framework that provides ready-made responsive grids, components, and utility classes for building modern, mobile-first websites quickly.

Introduction to Bootstrap

Bootstrap is a popular front-end framework used to design responsive and mobile-friendly websites quickly and efficiently. It was originally developed by Twitter and is now one of the most widely used CSS frameworks in web development.

Bootstrap contains pre-designed components, layout systems, utility classes, and JavaScript features that help developers create professional websites without writing large amounts of CSS from scratch.

According to Bootstrap's official documentation, the framework provides responsive design utilities, reusable components, and a powerful grid system that makes webpage development faster and easier. Bootstrap is especially useful for beginners because it simplifies webpage styling and layout creation.

What is a Front-End Framework?

A front-end framework is a collection of pre-written CSS and JavaScript code that helps developers build websites faster and more efficiently. Instead of writing every design feature manually, developers can use ready-made classes and components provided by frameworks such as Bootstrap.

Frameworks improve productivity because they provide reusable layouts, styling, and responsive utilities. They also help maintain consistency throughout a website.

Bootstrap is considered one of the most beginner-friendly frameworks because its documentation is well organized and easy to understand.

Using Bootstrap with CDN

Bootstrap can be added to a webpage using a CDN, which stands for Content Delivery Network. A CDN allows developers to use Bootstrap without downloading the framework files manually. 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

Using a CDN is useful because it makes setup faster and allows developers to start using Bootstrap classes immediately.

1. Bootstrap Grid System

One of the most important features of Bootstrap is its responsive grid system. The grid system allows developers to organize webpage content into rows and columns. Bootstrap uses a 12-column layout that automatically adjusts based on screen size.

<div class="row g-3">
    <div class="col-md-6 col-lg-4">Col 1</div>
    <div class="col-md-6 col-lg-4">Col 2</div>
    <div class="col-md-6 col-lg-4">Col 3</div>

The class .container centers content, .row creates a horizontal group of columns, and .col-md-* creates responsive columns. The .g-3 adds spacing between columns.

2. Responsive Design

Responsive design is one of Bootstrap's strongest features. Bootstrap includes predefined responsive classes that automatically adapt webpage layouts to different screen sizes. 

<div class="col-sm-12 col-md-6 col-lg-4">
    Responsive Column
</div>
Class Meaning Purpose
col-sm-12 Small screen Takes full width on smaller devices.
col-md-6 Medium screen Takes half width on medium devices.
col-lg-4 Large screen Takes one-third width on large devices.

3. Navigation Bars

Navigation bars are important because they help users move between different sections of a website. Bootstrap provides ready-made navigation bar components that are responsive and customizable. 

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Portfolio</a>
    </div>
</nav>

The navigation bar can automatically collapse into a mobile menu on smaller screens. This improves usability and mobile responsiveness.

5. Buttons

Bootstrap provides many pre-designed button styles that can be added using simple classes. Buttons are important because they help users interact with websites.

◈ Bootstrap Button  ·  HTML click a button ↴ 
<button class="btn btn-primary">Click Me</button>

Bootstrap includes different button styles such as primary, secondary, success, danger, and warning. These predefined styles save development time and improve consistency across webpages.

6. Forms

Forms are essential in websites because they allow users to enter information such as login details, contact information, and search queries. Bootstrap provides styled form components that improve appearance and usability. 

<div class="mb-3">
    <label class="form-label">Email address</label>
    <input type="email" class="form-control">
</div>


Bootstrap forms automatically include spacing, alignment, and responsive styling, making forms easier to create and more user-friendly.

7. Bootstrap Utility Classes

Bootstrap utility classes allow developers to quickly apply spacing, alignment, colors, typography, borders, display settings, and sizing without writing custom CSS. 

<div class="text-center p-4 bg-light">
    Utility class example
</div>

Spacing

m-3 adds margin and p-3 adds padding.

Colors

bg-primary and text-success apply theme colors.

Borders

border, rounded, and shadow-sm style containers.

Alignment

text-center aligns text in the center.

Display

d-flex activates flexible layout controls.

Utility classes help developers build webpages faster and reduce the amount of custom CSS required.

Bootstrap CSS Typography

Typography covers how text appears, including font size, weight, line height, alignment, and emphasis. Bootstrap supports semantic HTML elements and utility classes to quickly adjust text presentation.

Element or Class Description
h1 to h6 Heading levels from largest to smallest.
.display-1 to .display-6 Extra-large attention headings.
.lead Makes a paragraph stand out.
.fw-bold Applies bold font weight.
.text-center Centers text.
.text-uppercase Transforms text into uppercase.

Component Library Examples

Bootstrap includes a wide component library. Developers can use components such as navbars, dropdowns, buttons, cards, badges, alerts, and forms to build professional interfaces more quickly.

Navbar with Search

<nav class="navbar bg-body-tertiary">
    <div class="container-fluid">
        <a class="navbar-brand">Navbar</a>
        <form class="d-flex" role="search">
            <input class="form-control me-2" type="search" placeholder="Search">
            <button class="btn btn-outline-success" type="submit">Search</button>
        </form>
    </div>
</nav>

Live Dropdown Example

The dropdown below is fully functional thanks to Bootstrap's JavaScript bundle.

  • Action
  • Another action
  • Something else here
<div class="dropdown">
    <button class="btn btn-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown">
        Select an option
    </button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">Action</a></li>
        <li><a class="dropdown-item" href="#">Another action</a></li>
        <li><a class="dropdown-item" href="#">Something else here</a></li>
    </ul>

The dropdown component uses .dropdown, .dropdown-toggle, and .dropdown-menu. Bootstrap JavaScript enables the button to open and close the menu.
