JavaScript

JavaScript is a programming language used to add interactivity, dynamic behaviour, DOM updates, user events, and practical UI features to web pages.

Introduction To JavaScript

JavaScript is one of the most important programming languages used in modern web development. HTML creates the structure of a webpage, CSS controls the design, and JavaScript adds functionality so users can interact with the page.

JavaScript can be used to create image sliders, form validation, dropdown menus, calculators, games, pop-up messages, digital clocks, stopwatches, and many other interactive features.

JavaScript was originally created for web browsers, but today it is also used outside the browser with technologies such as Node.js. This means developers can use JavaScript for both front-end and back-end development.

Why JavaScript Is Important

JavaScript is important because it helps developers create websites that are more engaging and user friendly. Without JavaScript, websites would only display static information. JavaScript allows pages to react to user actions such as clicking buttons, typing into forms, or moving the mouse.

Form Validation

Checks user input before a form is submitted.

Real-Time Updates

Updates page content without reloading the browser.

Animations

Adds movement and visual effects to web pages.

Applications

Creates calculators, games, shopping carts, and chat apps.

API Data

Fetches data from external services and displays it on a page.

JavaScript And The DOM

The DOM stands for Document Object Model. It represents the structure of an HTML page as objects that JavaScript can access and modify. Using the DOM, developers can change text, add or remove elements, change styles, and respond to user actions. 

const heading = document.querySelector("h2");
heading.textContent = "Updated With JavaScript";

Event Handling In JavaScript

Events occur when users interact with a webpage. Examples include clicking buttons, typing in forms, and hovering over elements. JavaScript can detect these events and respond to them. 

button.addEventListener("click", function() {
  alert("Button clicked!");
});

JavaScript Objects

JavaScript exposes many built-in objects that model the page, the browser, and common data types. These objects provide useful properties and methods for web development.

Object Description Examples
documentRepresents the current HTML document.querySelector(), getElementById()
elementRepresents an HTML element.textContent, style, setAttribute()
windowRepresents the browser window or tab.innerWidth, setTimeout(), alert()
locationRepresents the current page URL.href, reload(), assign()
consoleUsed for logging and debugging.log(), warn(), error()
storageStores key-value data in the browser.setItem(), getItem(), removeItem()
arrayStores ordered lists of values.push(), map(), filter()
stringStores text values.length, includes(), trim()
numberRepresents numeric values.toFixed(), parseInt(), parseFloat()
mathProvides mathematical utilities.random(), floor(), round()
dateWorks with time and dates.Date.now(), getFullYear(), toLocaleString()
objectStores key-value data.Object.keys(), Object.values(), Object.entries()
promiseRepresents an asynchronous result.then(), catch(), Promise.all()
JSONWorks with structured data format.JSON.parse(), JSON.stringify()
